# PROPERTYHUB API
PropertyhubAPI is the backend for [Properyhub application](https://github.com/daktari01/propertyhub)

## Installation
1. Clone or download the repo `git clone https://github.com/daktari01/propertyhub-api.git`.
2. Run `rake db:setup` to create the database.
3. Run `bundle install` to install project dependencies.
4. Create a `.env` file and add `DEVISE_SECRET_KEY`. You can get your own by running `rake secret`
5. Run `rails server` to start the Rails server.
